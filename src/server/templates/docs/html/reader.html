Class for reading video stream frames and processing them using a producer and consumer.

Attributes:

__init__()[source]¶

Initialize the Reading class with necessary attributes.

The __init__ method sets up the Reading class with the producer, consumer, Redis connection, and VideoCapture object. These objects are necessary for reading and processing video stream frames.

__module__ = 'reader'¶
async __read_stream()¶

Read frames from the video stream and process them.

The read_stream method reads frames from the video stream, converts them to RGB format, encodes them as JPEG images, stores them in Redis, and sends them using the producer.

async __waiting_for_stream()¶

Continuously wait for video stream URLs from the consumer.

This method continuously polls the consumer for available video stream URLs. When a new URL is received, it updates the VideoCapture object with the new stream and checks the connection status.

async start()[source]¶

Start reading and processing the video stream frames.

The start method initiates the reading and processing of video stream frames. It creates and runs asynchronous tasks for waiting for stream URLs and reading the frames.
